# Project Brief

## Project Name

java-ai-rag-agent-platform

## Positioning

This project is a portfolio project for a Java backend developer transitioning into AI application engineering.

It demonstrates practical LLM application development using Spring Boot, Spring AI, RAG, Tool Calling, Agent workflows, and MCP.

## Target User

Enterprise users who need an AI knowledge base and workflow assistant.

## Core Product Goal

Build a Java-based AI knowledge base and agent workflow platform.

The system should eventually support:

- Document upload
- Document parsing
- Text chunking
- Embedding generation
- Vector storage
- Semantic search
- RAG question answering
- Source citations
- Chat history
- Streaming responses
- Tool calling
- Agent execution logs
- MCP integration

## Resume Positioning

Java backend developer transitioning into LLM application engineering.

Focus areas:

- Spring Boot
- Spring AI
- RAG
